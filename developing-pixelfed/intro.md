## Postavljanje razvoja
Samo za lokalnu / neproizvodnu upotrebu.
```bash
git clone https://github.com/pixelfed/pixelfed
cd pixelfed
composer install
php artisan install
php artisan serve

# Laravel development server started: http://127.0.0.1:8000
```
